# MintSplash
Change splash screen in Linux Mint

## Download
```
git clone https://github.com/lucasgabmoreno/mintsplash.git
cd mintsplash
```

## Change splash screen
Replace [image] with your 100x100px png image filename.
```
./mintsplash [imagefile]
```

## Restore default
```
./mintsplash restore
```
